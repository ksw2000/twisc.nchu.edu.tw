資通安全弱點通報機制(VANS)

C級學校聯合服務

為什麼學校要做 VANS？

因應資通安全管理法實施細則第六條第六款、第六條第七款，機關應盤點資通系統、建立相關風險評估機制。
依 110年8月23日 頒布之「資通安全責任等級分級辦法」修正條文，C級公務與非公務機關需於 民國112年8月23日 內完成 VANS 通報機制。

導入單位

C級學校-高中職
資訊組長 / 系統管理師 / 資安管理人員

C級學校聯合服務是什麼？

因應資通安全管理法實施，TWISC@NCHU（國立中興大學資通安全研究與教學中心）推出C級學校聯合服務計畫，整合產學技術提供符合行政院國家資通安全會報技術服務中心之 VANS 通報機制。

支援C級學校VANS導入與籌備之各項作業， 節省人工作業95%以上。

服務涵蓋

  • 雲端版系統

    雲端版運作，免除專責人員管理維運。

  • 弱點自動比對

    最即時的 CPE、NVD 字典和 CVE 清單資料檔案、安全性更新比對免煩惱。

  • 自動化盤點資通系統

    含硬體、軟體資訊，無論是 VANS 通報機制，或是校內年度資產盤點都能迅速產出資料，免除人工盤查作業與資料落差。

  • 排程定時通報

    串接行政院國家資通安全會報技術服務中心 VANS 通報平台 API 介面，達成自動化 VANS 整合通報作業。
